Projects

Calcver.Net

Automatic versioning using SemVer2 and Conventional Commits. This is a CLI packaged as a dotnet tool.

This tool can pick the right version based on commit messages and even generate change logs. This was designed with the Release Branches GitLab Flow in mind. Merges to master are treated as pre-releases and the tool automatically computes the appropriate release number. Stable releases must be manually tagged, using the pre-release numbers as a hint.

Konsole

Utility to write console applications WebHostBuilder style. Includes support for DI, Configuration, Options and Logging.

Cobalt

Very early work. Elm-inspired rust-wasm framework using wasm_bindgen. I'm writing this to learn more about rust and Elm at the same time.
